### [1-block_all_incoming_traffic_but](./1-block_all_incoming_traffic_but)
Configure ufw so that it blocks all incoming traffic, except the following TCP ports:
* 22 (SSH)
* 443 (HTTPS SSL)
* 80 (HTTP)
